# Лабораторна робота №9 - Авто майстерня Ford

## Варіант 5

База даних для авто майстерні Ford з використанням Docker + PostgreSQL + Python.

## Предметна область

Авто майстерня Ford (3 таблиці):
- **Клієнти** - інформація про клієнтів
- **Автомобілі** - марки Ford (fiesta, focus, fusion, mondeo)
- **Ремонти** - типи ремонту (гарантійний, плановий, капітальний)

## Структура проекту

```
├── docker-compose.yml    # Конфігурація Docker контейнерів
├── init_db.py           # Ініціалізація БД та створення таблиць
├── main.py              # Виведення таблиць та запитів
├── requirements.txt     # Python залежності
├── .gitignore          # Ігноровані файли
└── README.md           # Документація
```

## Запуск

### 1. Запуск Docker контейнерів

```bash
docker-compose up -d
```

### 2. Встановлення залежностей Python

```bash
pip install -r requirements.txt
```

### 3. Ініціалізація бази даних

```bash
python init_db.py
```

### 4. Запуск головної програми

```bash
python main.py
```

## SQL Запити

1. Гарантійні ремонти (сортування за клієнтом)
2. Вартість ремонту з урахуванням знижки (обчислювальне поле)
3. Ремонти для заданої марки авто (параметризований запит)
4. Загальна сума для кожного клієнта (підсумковий)
5. Кількість типів ремонту для клієнта (перехресний)
6. Кількість ремонтів для кожної марки
